php-fpm     ==> PHP FastCGI Process Manager (used with Nginx/Apache to serve PHP scripts).

php-curl    ==> Enables PHP to make HTTP requests.
php-gd      ==> Image processing library (resizing, cropping, etc.), needed by apps like WordPress.
php-xml     ==> XML parsing and manipulation support.
php-mbstring => Multibyte string functions (needed for UTF-8 text, non-English chars).
